# Gettysburg Mini Tactics

A lightweight, web-based, turn-based tactics prototype themed around the Battle of Gettysburg.

## Features
- 8x8 grid tactics
- Union vs Confederacy
- Move phase + attack phase
- High ground ("Little Round Top") bonus: +1 attack and +1 defense
- Event Cards system:
  - Draw 1 card per turn
  - Reinforcements: spawn an Infantry on your back row (if possible)
  - Ammo Shortage: -1 damage to your attacks for the current turn

## Run locally
Just open `index.html` in your browser.

## Run online
Just open https://lordskippythemagnificent.github.io
